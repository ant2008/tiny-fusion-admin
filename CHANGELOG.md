# Changelog
 所有更新在此文件中进行描述。
 
## [2.0.0](2023-11-09)

### Features
* 所使用核心框架(vue-element-plus-admin)已更新至于 2.3.0
* vue-element-plus-admin更改多项结构，其中调整最大的为对于slot的使用,直接可以在字段定义中书写slot。对于现有应用来说，还是采用原来模式但同时兼容两种使用方法。
* 对应用中的代码生成功能进行BUG修复以及增加相应的部分优化功能
* 对1.0版本中的部分架构用法进行了优化。
* 对1.0版本中的部分BUG进行修复。
